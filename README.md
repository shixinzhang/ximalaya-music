# 喜马拉雅 - 听歌 🎵

一个模拟喜马拉雅 FM "听歌"功能的网页应用，基于设计稿实现。

## 在线访问

🔗 **https://[your-github-username].github.io/ximalaya-music/**

## 功能特性

### 核心功能
- ✅ 新人引导页面
- ✅ 底部导航栏（首页/喜欢/我的/VIP/更多）
- ✅ 顶部主题化 Tab 栏（推荐/经典老歌/流行热歌/助眠音乐/自然白噪音/动感DJ/健身/禅意养生）
- ✅ 双列歌单展示
- ✅ 播放器页面（大封面 + 歌词滚动高亮 + 播放控制）
- ✅ 心动模式（基于点赞的个性化推荐）
- ✅ 循环模式设置（列表播放/单曲循环/随机播放）

### 播放器特性
- 🎵 歌词随播放进度自动滚动高亮
- 🔄 封面旋转动画（播放时）
- ⏱️ 进度条拖拽控制
- ❤️ 点赞收藏功能
- 🔄 播放模式切换

### 技术栈
- HTML5
- CSS3（Flexbox + Grid）
- Vanilla JavaScript
- LocalStorage 数据持久化
